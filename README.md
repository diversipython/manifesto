# Sobre o DiversiPy

### Do que se trata

O DiversiPy foi idealizado durante a Python Brasil 2022, que foi realizada em Manaus-AM. O anúncio foi feito durante uma palestra relâmpago e várias pessoas se inscreveram em um grupo do Telegram.

Havia uma demanda da comundidade para uma rede de acolhimento e debate para as demandas e necessidades da comunidade LGBTQIAP+ dentro do ecossistema Python brasileiro.

### Objetivos

Prover espaços, materiais e debates da população LGBTQIAP+ visando fortalecer a diversidade e prover conscientização dessas demandas e soluções para a comunidade pythônica brasileira.

### Missão

Temos como missão "Prover um ambiente seguro para o debate e acolhimento da população LGBTQIAP+ dentro do contexto da comunidade Python brasileira."

### Valores

- Diversidade
- Empatia
- Alteridade
- Ética
- Responsabilidade social
- Democratização da informação
